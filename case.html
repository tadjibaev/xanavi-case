1. Краткое описание инцидента

Суть: Неустановленные лица осуществили несанкционированное копирование (клонирование лицензии) коммерческого приложения «Xanavi Voice» (пакет com.ulusoft.bydvoice) и организовали его незаконное распространение через мессенджер Telegram для установки на головные устройства автомобилей BYD.

Что было похищено

Объект Описание Критичность
Лицензионные данные USER_ID, токены авторизации, ключи шифрования из SharedPreferences КРИТИЧЕСКАЯ
APK-файл приложения voice.apk (289 МБ), защищён Virbox Protector КРИТИЧЕСКАЯ
Базы данных команд commands_ru.sqlite, commands_tr.sqlite, commands_uz.sqlite, commands_ky.sqlite ВЫСОКАЯ
Файлы DRM-защиты Virbox Protector: l5e7186e3.so, vbp.dex ВЫСОКАЯ
Azure API-ключ Ключ подписки Cognitive Services TTS (захардкожен в приложении) ВЫСОКАЯ

Метод атаки

Клонирование лицензии через ADB с root-доступом к головному устройству BYD. Код приложения не был модифицирован — атакующие скопировали данные авторизации с легитимного устройства и распространяют инструкции по их установке на другие устройства.

3.1. Анализ инструментов атаки

Файл: install_hack.txt — скрипт установки APK

Тип: Python-скрипт
Функция: Замена системного APK в /system/priv-app/BydVoice/BydVoice.apk
Метод: ADB root + remount → копирование через /data/local/tmp
Особенности: Создаёт резервную копию оригинала (backup), комментарии на русском языке 
# Ключевые команды из скрипта атаки:
adb root
adb remount
adb push "voice.apk" /data/local/tmp/voice.apk
adb shell cp /data/local/tmp/voice.apk /system/priv-app/BydVoice/BydVoice.apk
adb shell chmod 644 /system/priv-app/BydVoice/BydVoice.apk

Тип: Windows Batch-скрипт (.bat)

4.3. SQL-запросы для выявления всех скомпрометированных аккаунтов

Для серверной команды: Выполнить на базе данных rtexecute.com 
-- 1. Один USER_ID с нескольких IP-адресов (последние 48 часов)
SELECT user_id, COUNT(DISTINCT ip_address) AS ip_count,
       GROUP_CONCAT(DISTINCT ip_address) AS ips
FROM api_logs
WHERE endpoint IN ('/switch-check-up', '/assistant/stream')
  AND timestamp > NOW() - INTERVAL 48 HOUR
GROUP BY user_id
HAVING ip_count > 2
ORDER BY ip_count DESC;

-- 2. Новые IP-адреса после кражи (6 февраля 2026)
SELECT user_id, ip_address, user_agent, timestamp
FROM api_logs
WHERE timestamp > '2026-02-06 00:00:00'
  AND ip_address NOT IN (
    SELECT DISTINCT ip_address FROM api_logs
    WHERE user_id = api_logs.user_id
      AND timestamp < '2026-02-06 00:00:00'
  );

-- 3. Параллельные SSE-подключения (один userId = одно устройство)
SELECT user_id, COUNT(*) AS active_connections
FROM active_sse_connections
GROUP BY user_id
HAVING active_connections > 1;

-- 4. Конкретный украденный USER_ID — все обращения
SELECT ip_address, endpoint, timestamp, user_agent
FROM api_logs
WHERE user_id = '$2b$12$enTfKDnfr7Kk77rY/zuYuOCLFxG2nMltGxzjKy5YywrR.cOBKgKom'
ORDER BY timestamp DESC
LIMIT 1000;

5. Правовая база Для юристов

Важно: Исходный код программы для ЭВМ в странах СНГ защищается авторским правом (как литературное произведение), а не патентом. Программы «как таковые» не являются патентоспособными объектами. Авторское право возникает автоматически с момента создания произведения, но добровольная государственная регистрация значительно упрощает доказывание в суде.

8.2. Механизм еженедельного kill-switch

Встроенный механизм самоуничтожения пиратской копии:

Каждое воскресенье в 5:10 (WeeklyTaskManager) приложение:
  1. blockApp() → удаляет все базы данных (localCache.deleteDatabases())
  2. Отправляет запрос к /switch-check-first для скачивания новых баз
  3. Если сервер возвращает 403 → базы не скачаны → unblockApp() не вызывается
  4. Приложение навсегда заблокировано с экраном «Пожалуйста подождите...»

Максимальное время жизни пиратской копии: 7 дней.

8.3. Пример серверного кода блокировки

// Node.js / Express — middleware для всех эндпоинтов

const blockedUserIds = new Set([
  '$2b$12$enTfKDnfr7Kk77rY/zuYuOCLFxG2nMltGxzjKy5YywrR.cOBKgKom',
  // ... другие скомпрометированные userId
]);

function checkBlockedUser(req, res, next) {
  const userId = req.body?.userId || req.query?.userId;

  if (userId && blockedUserIds.has(userId)) {
    // Логируем для доказательной базы
    logger.warn('BLOCKED pirate access', {
      userId,
      ip: req.ip,
      endpoint: req.path,
      userAgent: req.headers['user-agent'],
      timestamp: new Date().toISOString()
    });

    return res.status(403).json({
      error: 'access_denied',
      message: 'Доступ заблокирован. Использование контрафактного ПО '
             + 'преследуется по ст. 149-1 УК РУз.'
    });
  }

  next();
}

// Применить ко всем эндпоинтам
app.use('/switch-check-up', checkBlockedUser);
app.use('/switch-check-first', checkBlockedUser);
app.use('/switch-check', checkBlockedUser);
app.use('/assistant/stream', checkBlockedUser);
app.use('/car-token', checkBlockedUser);
app.use('/car-sse', checkBlockedUser);
